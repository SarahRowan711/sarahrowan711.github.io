Spooky Eyes Animation Project

This project was an exercise intended to familiarize us with manipulating objects in the DOM using javascript. This is a simple animation which lets you move the irises around in a set of creepy eyes! Moving the mouse around in a browser moves the yellow irises. I modified the colors and the shape of the eyes from the original project.
